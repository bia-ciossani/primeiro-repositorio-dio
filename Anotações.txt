Principais Comandos GIT BASH

mkdir (make directory) - abrir um novo diretório.
ls (list) - listar as pastas.
cd - entrar na pasta x.
git init - criar diretório
ls -a - abrir "pasta oculta" do diretório.
cd .. - voltar um nível de pasta.
git clone + link do repositório no github - clonar o repositório.
git pull - atualizar alteração no repositório online para a máquina.
git push - atualizar da máquina para o repositório online.
git status - mostra o situação
git commit - "confirma".
